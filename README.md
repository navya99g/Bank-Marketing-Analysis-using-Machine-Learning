# Bank Marketing Campaign Analysis using Machine Learning

## Problem Statement
Predict whether a bank customer will subscribe to a term deposit based on demographic and marketing campaign data.

## Dataset
Bank Marketing dataset

## Project Objective
- Analyse customer behaviour
- Identify factors influencing subscription
- Build and evaluate classification models

## Approach
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Encoding categorical variables
4. Model building using Logistic Regression and tree-based models
5. Model evaluation using precision, recall and F1-score

## Results
The model helps identify potential customers more likely to subscribe, supporting marketing decisions.

## Tools & Technologies
Python, Pandas, NumPy, Scikit-learn, Machine Learning
